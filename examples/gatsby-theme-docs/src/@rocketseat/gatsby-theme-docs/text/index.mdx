# المقدمة

أهلًا بك في دليل أسلوب كتابة المحتوى (Content Style Guide) - الإصدارة الأولى، هذا الدليل (مفتوح المصدر) مساهمة لإثراء مجال كتابة تجربة المستخدم (UX Wriitng) باللغة العربية، يمكنك المساهمة معنا في إضافة أو تعديل وكذلك تحسين الدليل بإتباع التعليمات على Github.

## كيف تستخدم هذا الدليل؟

- يمكنك البدء في استخدام هذا الدليل بالاطلاع على الأقسام في القائمة الرئيسية وتطبيق ما يلزم على مشروعك
- يتم تحديث الدليل باستمرار، استعرض سجّل التغييرات - [سجل التغييرات](https://github.com/UX-Writing/Content-Style-Guide-in-Arabic/releases/)
 وما تم إضافته أو تغيير وتعديله، احرص على متابعة التغييرات باستمرار إذا اعتمدت الدليل في مشروعك.
